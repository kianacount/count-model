What is CountGD?

CountGD is an open-world multi-modal object counting model that flexibly counts any type of object through prompts and examples. It generalizes across new domains without retraining, achieving high accuracy even on complex and crowded images.

Key Innovations

CountGD is the first model to combine both visual exemplars and text prompts to guide object counting, leveraging the GroundingDINO foundation. It sets a new benchmark in open-world counting, enabling users to count any object without retraining, while offering enhanced accuracy and flexibility through multi-modal prompting.

Clarification

CountGD is a counting model — it outputs the number of instances of an object. It does not perform segmentation or provide object masks. The focus is purely on accurate counting based on the user’s text or visual prompt.
